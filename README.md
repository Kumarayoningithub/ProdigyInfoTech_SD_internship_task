# Task-1
# Temperature Converter (Java Swing)

A simple **Temperature Converter Desktop Application** built using **Java Swing**.  
This application allows users to convert temperature values between **Celsius, Fahrenheit, and Kelvin** through an interactive graphical interface.

---

# Project Overview

The **Temperature Converter** is a GUI-based Java application that takes a temperature input from the user and converts it into the other temperature units.

The program supports conversions between:

- Celsius
- Fahrenheit
- Kelvin

It demonstrates **Java GUI development, event handling, and mathematical calculations**.

---

# Features

- 🌡️ Convert temperature between **Celsius, Fahrenheit, and Kelvin**
- 🖥️ User-friendly **Graphical User Interface**
- 📥 Input temperature using a text field
- 🔽 Select temperature unit using dropdown menu
- 🔄 Instant conversion using a button
- ⚠️ Error handling for invalid input
- 📊 Displays all converted values simultaneously

---

# Technologies Used

- Java
- Java Swing
- AWT
- Event Handling (ActionListener)
- Basic Mathematical Operations

---

# Learning Outcomes

Through this project, I practiced:

- Building GUI applications using **Java Swing**
- Handling user actions using **ActionListener**
- Implementing **temperature conversion formulas**
- Managing **user input and validation**
- Designing simple **interactive desktop applications**

# Task-2
# Number Guessing Game (Java Swing)

A simple **Number Guessing Game** built using **Java Swing GUI**.  
The player must guess a randomly generated number between **1 and 100**, and the program provides hints whether the guess is **too high or too low** until the correct number is guessed.

---

# Project Overview

This project is a **desktop GUI application** that demonstrates basic Java concepts such as:

- Java Swing GUI development
- Event handling
- Random number generation
- User input validation
- Game logic implementation

The application keeps track of the **number of attempts** and allows the user to **restart the game** after winning.

---

# Features

- 🎮 Interactive GUI using **Java Swing**
- 🔢 Random number generated between **1 – 100**
- ⬆️ Hint system (Too High / Too Low)
- 📊 Tracks number of attempts
- 🔁 Restart button to play again
- ⚠️ Input validation for incorrect entries
- 🖥️ Responsive window centered on screen

---

# Technologies Used

- Java
- Java Swing
- AWT
- Event Handling
- Random Class

---

# How the Game Works

1. The program generates a **random number between 1 and 100**.
2. The player enters a number in the input field.
3. The program checks the guess:
   - If the number is **greater**, it shows **"Too High!"**
   - If the number is **smaller**, it shows **"Too Low!"**
4. When the correct number is guessed:
   - A **success message appears**
   - Total attempts are displayed
5. The player can click **Restart** to play again.
